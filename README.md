### 安装
```
git clone https://github.com/Joeays/vue-admin.git

//先git克隆或者下载下来，进入项目目录执行npm命令

npm install
```
#### Node.js要求：
基于 Koa2, 使用了异步函数，其中Node.js版本推荐 8.11.0+
